## О проекте

Современная страница обратной связи на чистом HTML, CSS и Vanilla JavaScript. Встроенная карта Google Maps, валидация формы и toast-уведомление об успешной отправке. Сборка не требуется.

---

## Возможности

- Встроенная карта Google Maps с информационной плашкой (адрес, телефон, email)
- Форма с полями: имя, email, телефон и сообщение
- Валидация: подсветка незаполненных полей при отправке
- Toast-уведомление об успешной отправке
- Современный дизайн: градиентный фон, карточка с тенью, плавные анимации
- Полностью адаптивная вёрстка: десктоп, планшет, мобильный
- Шрифт Inter через Google Fonts

## Быстрый старт

Сборка не требуется — откройте `index.html` напрямую или запустите локальный сервер:

```bash
git clone https://github.com/Topchikdan/Business-card-website-HTML.git
cd Business-card-website-HTML
npx serve .
```
