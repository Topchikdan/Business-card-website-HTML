# Contact Us — HTML Page
**Современная страница обратной связи с картой Google Maps, валидацией формы и адаптивной вёрсткой — без фреймворков.**

## ✨ Возможности

- 🗺 Встроенная карта Google Maps с информационной плашкой (адрес, телефон, email)
- 📝 Форма с полями: имя, email, телефон и сообщение
- ✅ Валидация — подсветка незаполненных полей при отправке
- 🔔 Toast-уведомление об успешной отправке
- 🎨 Современный дизайн: градиентный фон, карточка с тенью, плавные анимации
- 📐 Полностью адаптивная вёрстка (десктоп → планшет → мобильный)
- 🔠 Шрифт Inter через Google Fonts

## 🛠 Технологии

- **HTML5** — семантическая разметка, атрибуты доступности
- **CSS3** — Flexbox, градиенты, переходы, медиазапросы
- **Vanilla JavaScript (ES6+)** — валидация, toast-уведомление, очистка формы
- **Google Maps Embed API** — карта без ключа

## 📁 Структура проекта

```
contact-page-html/
├── index.html    # Разметка страницы и inline-скрипт
└── styles.css    # Все стили, анимации и адаптивность
```

## 🚀 Как запустить

Сборка не нужна — просто открой файл в браузере:

# Клонировать репозиторий
git clone https://github.com/your-username/contact-page-html.git

# Перейти в папку
cd contact-page-html

# Открыть в браузере
open index.html

Или перетащи `index.html` прямо в браузер — и всё готово.
